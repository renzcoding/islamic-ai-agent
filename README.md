# Nuansa AI - Islamic Smart Assistant 🌙

Nuansa AI adalah aplikasi AI Agent Islami interaktif yang dibangun menggunakan **Streamlit** dan ditenagai oleh **Google Gemini Pro API**. Aplikasi ini dirancang untuk menjadi teman diskusi yang hangat, santun, dan interaktif dalam membahas topik produktivitas islami, motivasi ibadah, dan adab harian.

## 🛠️ Cara Menjalankan di Lokal

1. Clone repositori ini:

```bash
   git clone [https://github.com/renzcoding/islamic-ai-agent.git](https://github.com/renzcoding/islamic-ai-agent.git)
   cd islamic-ai-agent
```
